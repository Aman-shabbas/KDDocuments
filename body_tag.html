< body >

The < body > tag in HTML defines the main content of an HTML document, encompassing all visible content on a webpage, including text, images, videos, and other multimedia elements1. It acts as a container for all the visible elements, providing structure and organization to the webpage's content13.

Attributes

  1. bgcolor

    Sets the background color of the page

    Examples
    • < body bgcolor="lightblue" >

      sets the background color of the entire HTML document

    • < body bgcolor="#FFFF00" >

      sets the background color to yellow using a hexadecimal color code.

  2. text

    Sets the color of the text on the webpage

    Examples
    • < body text="#008000" >

      Sets the default text color for the entire document

  3. link

    Sets the color of unvisited links

    Examples
    • < body link="color_name" >

      Sets the default text color for the entire document

  4. alink

    Sets the color of active links

    Examples
    • <body alink="green">

      specifies the color of an active link in a document

  5. vlink

    Sets the color of visited links

    Examples
    • <body vlink="red">

      sets the color of visited links to red

  6. background

    Specifies an image to use as the background
