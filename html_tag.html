< html >

The < html > tag signifies the root of an HTML document, informing browsers that everything within it should be interpreted as HTML-coded information

Attributes

  1. lang

    to represent specify the language of documents

    Examples
    • < html lang="en" >

      specifies English as the language.

    • < html lang="en-AU" >

      specifies the language as English in Australia using country code

  2. dir

    to represent the text direction of the content within the HTML Document

    Examples
    • < html dir="rtl" >

      Specifies that the text direction should be right-to-left

    • < html dir="ltr" >

      Specifies that the text direction should be left-to-right(default)
